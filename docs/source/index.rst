
Overview of INCITE data obtained by Rayleigh
---------------------------------------------
Here is a documentation of data for magnetohydorodynamics (MHD) simulations sets generated by the DOE INCITE project.
Data for each simulation is under a certain directory. Parameters for each simulation is described in the file "main_input".

INCITE data consists of three targets: Geodynamo, Jovian atomosphere, Dynamos in Icy giants. The paramteres and visualization examples for each models are listed in each sections.
