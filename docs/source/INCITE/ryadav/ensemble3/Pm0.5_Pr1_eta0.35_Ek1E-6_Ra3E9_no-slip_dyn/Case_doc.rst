Directory.
=========================================
Data is stored in the following directory:

- `ryadav/ensemble3/Pm0.5_Pr1_eta0.35_Ek1E-6_Ra3E9_no-slip_dyn <https://incite.geodynamics.org/DATA/ryadav/ensemble3/Pm0.5_Pr1_eta0.35_Ek1E-6_Ra3E9_no-slip_dyn>`_

Available data.
==========================================

The following data for analysis are aviable:

- Parameter file (original): `main_input.org <https://incite.geodynamics.org/DATA/ryadav/ensemble3/Pm0.5_Pr1_eta0.35_Ek1E-6_Ra3E9_no-slip_dyn/main_input.org>`_
- Checkpoint data directory: `Checkpoints <https://incite.geodynamics.org/DATA/ryadav/ensemble3/Pm0.5_Pr1_eta0.35_Ek1E-6_Ra3E9_no-slip_dyn/Checkpoints>`_
- Global averaged data directory: `G_Avgs <https://incite.geodynamics.org/DATA/ryadav/ensemble3/Pm0.5_Pr1_eta0.35_Ek1E-6_Ra3E9_no-slip_dyn/G_Avgs>`_
- Sphere averaged data directory: `Shell_Avgs <https://incite.geodynamics.org/DATA/ryadav/ensemble3/Pm0.5_Pr1_eta0.35_Ek1E-6_Ra3E9_no-slip_dyn/Shell_Avgs>`_
- Shell spectrum data directory: `Shell_Spectra <https://incite.geodynamics.org/DATA/ryadav/ensemble3/Pm0.5_Pr1_eta0.35_Ek1E-6_Ra3E9_no-slip_dyn/Shell_Spectra>`_
- Longitudinal averaged data directory: `AZ_Avgs <https://incite.geodynamics.org/DATA/ryadav/ensemble3/Pm0.5_Pr1_eta0.35_Ek1E-6_Ra3E9_no-slip_dyn/AZ_Avgs>`_
- Fields on spheres directory: `Shell_Slices <https://incite.geodynamics.org/DATA/ryadav/ensemble3/Pm0.5_Pr1_eta0.35_Ek1E-6_Ra3E9_no-slip_dyn/Shell_Slices>`_
- Equatorial fields directory: `Equatorial_Slices <https://incite.geodynamics.org/DATA/ryadav/ensemble3/Pm0.5_Pr1_eta0.35_Ek1E-6_Ra3E9_no-slip_dyn/Equatorial_Slices>`_
