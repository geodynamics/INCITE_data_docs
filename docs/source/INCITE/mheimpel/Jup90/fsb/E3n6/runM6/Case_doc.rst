J90-pn2-ht4-E3n6-M6
-----------------------------------------------------------

Directory.
=========================================
Data is stored in the following directory:

- `mheimpel/Jup90/fsb/E3n6/runM6 <https://incite.geodynamics.org/DATA/mheimpel/Jup90/fsb/E3n6/runM6>`_

Available data.
==========================================

The following data for analysis are aviable:

- Parameter file (original): `main_input.org <https://incite.geodynamics.org/DATA/mheimpel/Jup90/fsb/E3n6/runM6/main_input.org>`_
- Checkpoint data directory: `Checkpoints <https://incite.geodynamics.org/DATA/mheimpel/Jup90/fsb/E3n6/runM6/Checkpoints>`_
- Global averaged data directory: `G_Avgs <https://incite.geodynamics.org/DATA/mheimpel/Jup90/fsb/E3n6/runM6/G_Avgs>`_
- Sphere averaged data directory: `Shell_Avgs <https://incite.geodynamics.org/DATA/mheimpel/Jup90/fsb/E3n6/runM6/Shell_Avgs>`_
- Shell spectrum data directory: `Shell_Spectra <https://incite.geodynamics.org/DATA/mheimpel/Jup90/fsb/E3n6/runM6/Shell_Spectra>`_
- Longitudinal averaged data directory: `AZ_Avgs <https://incite.geodynamics.org/DATA/mheimpel/Jup90/fsb/E3n6/runM6/AZ_Avgs>`_
- Fields on spheres directory: `Shell_Slices <https://incite.geodynamics.org/DATA/mheimpel/Jup90/fsb/E3n6/runM6/Shell_Slices>`_
